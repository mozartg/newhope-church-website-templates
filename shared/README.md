# Shared Images

Upload your church images here. The templates reference these files:

| File | Used For | Dimensions |
|------|----------|------------|
| `pastor.jpg` | Hero section on all 15 templates | 3:4 portrait |
| `firstlady.png` | Pastor bio page | 3:4 portrait |
| `congregation.jpg` | Background for Mission/About sections | 16:9 landscape |
| `smallgroup.jpg` | Small groups / ministries section | 16:9 landscape |
| `worship.jpg` | Worship / events section | 16:9 landscape |

## How to Upload

1. Go to https://github.com/mozartg/newhope-church-website-templates
2. Click on the `shared/` folder
3. Click **"Add file"** → **"Upload files"**
4. Drag and drop your images (make sure filenames match the table above)
5. Click **"Commit changes"**

## Image Tips

- **Pastor photo**: Professional portrait, warm lighting, welcoming smile
- **First Lady**: Elegant portrait, coordinated with pastor's style
- **Congregation**: Wide shot of people during service, showing community
- **Small Group**: Intimate Bible study or fellowship setting
- **Worship**: Choir or congregation in praise, dynamic and uplifting

All images should be JPG format, optimized for web (under 200KB each recommended).
