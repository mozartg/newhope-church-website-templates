## Quick Start (2 Minutes)

### Step 1: Enable GitHub Pages
1. Go to **Settings** → **Pages** in this repository
2. Under "Source", select **Deploy from a branch**
3. Select **main** branch and **/(root)** folder
4. Click **Save**
5. Wait 2-3 minutes for the site to go live at:
   `https://mozartg.github.io/newhope-church-website-templates`

### Step 2: Add Your Images
1. Go to the `shared/` folder in this repository
2. Click **Add file** → **Upload files**
3. Upload these 5 images (filenames must match exactly):
   - `pastor.jpg` — Pastor portrait photo
   - `firstlady.png` — First Lady portrait photo
   - `congregation.jpg` — Congregation/wide shot
   - `smallgroup.jpg` — Small group Bible study
   - `worship.jpg` — Worship service photo
4. Click **Commit changes**

That's it! Your website is live.

## For Churches & Virtual Assistants: How to Edit

### Change Text
Open any template in a text editor. Search for what you want to change:

| What to Change | Search For |
|----------------|-----------|
| Church Name | "New Hope Community Church" |
| Service Times | "10:00 AM", "7:00 PM", "6:00 PM" |
| Address | "123 Faith Avenue, Rochester NY 14604" |
| Phone | "(585) 555-0123" |
| Email | "hello@newhopechurch.org" |
| Pastor Name | "Pastor James Williams" |
| First Lady Name | "Michelle Williams" |

### Change Colors
Each template has color codes at the top of the CSS:
- `--primary:` — Main brand color
- `--accent:` — Gold/highlight color
- `--text:` — Text color
- `--bg:` — Background color

### Replace Images
Upload new images to the `shared/` folder (same filenames). All 15 templates will automatically use them.

### Add Your Own Pages
Copy any `index.html` from a variation folder, rename it, and edit. Link between pages using regular `<a href="...">` tags.
