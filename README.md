## For Churches & Virtual Assistants: How to Edit

### 1. Pick a Design
Open `index.html` in your browser to see all 15 options. Click any to preview.

### 2. Customize the Text
Each template is a single HTML file. Open it in any text editor and search for the text you want to change:

- **Church Name:** Find "New Hope Community Church"
- **Service Times:** Find "10:00 AM", "7:00 PM", "6:00 PM"
- **Address:** Find "123 Faith Avenue, Rochester NY 14604"
- **Phone:** Find "(585) 555-0123"
- **Email:** Find "hello@newhopechurch.org"
- **Pastor Name:** Find "Pastor James Williams" and "Michelle Williams"

### 3. Change Colors
Each template has CSS color codes at the top. Look for:
- `background:` for the main color
- `color:` for accent colors

### 4. Replace Images
Put your own images in the `shared/` folder:
- `pastor.jpg` - Pastor photo
- `firstlady.png` - First Lady photo
- `congregation.jpg` - Congregation background
- `smallgroup.jpg` - Small group background
- `worship.jpg` - Worship background

### 5. Go Live
Upload the files to any web host:
- **GitHub Pages** (free)
- **Netlify** (free, drag & drop)
- **Vercel** (free)
- **Bluehost/HostGator** (traditional hosting)
